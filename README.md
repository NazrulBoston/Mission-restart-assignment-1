
## What is the difference between null and undefined?
Ans: Undefined means a variable was created but not given a value. Null means the value is empty on purpose. 
Undefined is automatic, null is set by the programmer.


## What is the use of the map() function in JavaScript? How is it different from forEach()?
Ans:The map() function creates and returns a new array after changing each item.
The forEach() function just loops through the array but does not return anything.
Use map when you want a new array.


## What is the difference between == and ===?
Ans: == checks only value and can change data type.
=== checks value and type without changing anything.
=== is safer and better to use.


## What is the significance of async/await in fetching API data?
Ans: Async/await makes API calls easier to read and write.
It waits for the result before moving to the next line. It makes code cleaner.


## Explain the concept of Scope in JavaScript (Global, Function, Block).
Ans: Scope means where a variable can be used.
Global scope works everywhere.
Function scope works only inside a function.
Block scope works only inside { }.
